# AI 기반 다국어 음성 합성 및 실시간 립싱크 더빙 시스템 개발일지

## 📋 오늘의 작업 내용

### 1. 시스템 아키텍처 설계

- **모듈형 구조 (Modular Architecture)** 채택.
- 각 기능(STT, TTS, VC 등)을 독립적인 모듈로 개발하여 유지보수성 향상.
- **Backend**: FastAPI를 사용하여 각 모듈을 API로 래핑.
- **Frontend**: Streamlit을 사용하여 빠른 프로토타이핑 및 사용자 인터페이스 제공.

### 2. 데이터 파이프라인 설계

- `Input Video` -> `Audio Extraction` -> `STT` -> `Translation` -> `TTS` -> `RVC` -> `Wav2Lip` -> `Output Video`.
- 중간 산출물(오디오 파일, 텍스트 파일)은 `data/` 디렉토리에 구조화하여 저장.
