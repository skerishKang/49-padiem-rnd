# AI 기반 다국어 음성 합성 및 실시간 립싱크 더빙 시스템 개발일지

## 📋 오늘의 작업 내용

### 1. FFmpeg 설치 및 설정

- 영상 처리를 위한 필수 도구인 FFmpeg 설치.
- 시스템 환경 변수(PATH)에 등록하여 터미널에서 접근 가능하도록 설정.

### 2. 오디오 추출 스크립트 작성

- `modules/audio_extractor.py` (가칭) 작성.
- 비디오 파일에서 오디오를 추출하여 WAV 또는 MP3로 저장하는 기능 구현.
- **옵션**: 샘플 레이트(16k, 44.1k), 채널(Mono/Stereo) 설정 가능하도록 설계.

## 🔧 기술적 진행사항

### Python Subprocess

```python
import subprocess
cmd = f"ffmpeg -i {input_video} -vn -acodec pcm_s16le -ar 16000 -ac 1 {output_audio}"
subprocess.run(cmd, shell=True)
```
