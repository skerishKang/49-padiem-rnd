# AI 기반 다국어 음성 합성 및 실시간 립싱크 더빙 시스템 개발일지

## 📋 오늘의 작업 내용

### 1. Whisper 연동

- `faster-whisper` 라이브러리 설치.
- 어제 추출한 오디오 파일을 입력으로 넣어 텍스트 변환 테스트.

### 2. 자막 포맷팅

- Whisper 결과(Segments)를 파싱하여 SRT(SubRip Subtitle) 형식으로 변환하는 로직 구현.
- 타임스탬프(Start, End) 포맷팅 (`00:00:00,000`).

## 🔧 기술적 진행사항

### Whisper Inference

```python
from faster_whisper import WhisperModel
model = WhisperModel("large-v3", device="cuda", compute_type="float16")
segments, info = model.transcribe("audio.wav", beam_size=5)
```
