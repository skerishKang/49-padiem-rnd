# AI 기반 다국어 음성 합성 및 실시간 립싱크 더빙 시스템 개발일지

## 📋 오늘의 작업 내용

### 1. 오디오 정규화 (Normalization)

- 분리된 보컬의 볼륨이 너무 작거나 큰 경우를 대비해 -20dB LUFS 표준으로 맞추는 작업.
- `pydub` 라이브러리 활용.

### 2. 무음 제거 (Silence Removal)

- STT 처리 효율을 높이기 위해 앞뒤의 긴 무음 구간(Silence)을 트리밍(Trimming).

## 🔧 기술적 진행사항

### Pydub Usage

```python
from pydub import AudioSegment, effects
sound = AudioSegment.from_file("vocals.wav")
normalized_sound = effects.normalize(sound)
```
