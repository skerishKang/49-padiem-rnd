# AI 기반 다국어 음성 합성 및 실시간 립싱크 더빙 시스템 개발일지

## 📋 오늘의 작업 내용

### 1. 환각(Hallucination) 현상 분석

- Whisper 모델이 무음 구간이나 잡음 구간에서 "MBC 뉴스", "시청해주셔서 감사합니다" 등의 텍스트를 반복 생성하는 현상 확인.
- 이는 학습 데이터의 편향 때문.

### 2. 해결 방안 (Prompt Engineering)

- Whisper의 `initial_prompt` 옵션 활용.
- "이 영상은 회의 녹음입니다. 대화 내용만 받아적으세요." 등의 프롬프트 주입 테스트.
- **결과**: 일부 개선되었으나 완벽하지 않음.

## 🔧 기술적 진행사항

### Prompting

```python
model.transcribe("audio.wav", initial_prompt="...")
```
