# AI 기반 다국어 음성 합성 및 실시간 립싱크 더빙 시스템 개발일지

## 📋 오늘의 작업 내용

### 1. 반복 텍스트 제거

- 동일한 문구가 연속으로 3회 이상 등장하면 제거하는 로직 구현.
- 정규표현식 활용.

### 2. 비속어 및 특수문자 필터링

- 불필요한 특수문자나 이모지 제거.
- `clean_text()` 함수 작성하여 `modules/text/cleaner.py`에 저장.

## 🔧 기술적 진행사항

### Regex

```python
import re
text = re.sub(r'(.)\1{3,}', r'\1', text) # 반복 문자 축소
```
