# AI 기반 다국어 음성 합성 및 실시간 립싱크 더빙 시스템 개발일지

## 📋 오늘의 작업 내용

### 1. 번역 모듈 구조화

- `modules/translation/translator.py` 작성.
- `DeepTranslator` 클래스 구현.
- 소스 언어 자동 감지(`auto`) 옵션 활용.

### 2. 자막 번역 로직

- SRT 파일을 파싱하여 텍스트 부분만 추출 -> 번역 -> 다시 SRT 구조로 결합.
- 타임스탬프는 그대로 유지.

## 🔧 기술적 진행사항

### Batch Translation

- 한 문장씩 번역하면 API 호출 제한에 걸릴 수 있음. -> 여러 문장을 `\n`으로 합쳐서 한 번에 보내고 다시 나누는 배치 처리 방식 고려 중.
