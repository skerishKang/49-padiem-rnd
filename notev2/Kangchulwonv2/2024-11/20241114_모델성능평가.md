# AI 기반 다국어 음성 합성 및 실시간 립싱크 더빙 시스템 개발일지

## 📋 오늘의 작업 내용

### 1. 다국어 번역 테스트

- 한국어 -> 영어, 일본어, 중국어(간체), 스페인어, 프랑스어 테스트.
- **결과**: 영어, 일본어는 품질 우수. 중국어는 고유명사 처리에서 가끔 오류.

### 2. 용어집(Glossary) 기능 기획

- 특정 단어(예: 회사명, 제품명)를 지정된 단어로 번역하도록 강제하는 기능 필요성 확인.
- 단순 치환(Replace) 방식으로 1차 구현 예정.

## 🔧 기술적 진행사항

### Dictionary Replacement

```python
glossary = {"파디엠": "Padiem", "더빙": "Dubbing"}
for k, v in glossary.items():
    text = text.replace(k, v)
```
