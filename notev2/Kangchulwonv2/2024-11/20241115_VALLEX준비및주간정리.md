# AI 기반 다국어 음성 합성 및 실시간 립싱크 더빙 시스템 개발일지

## 📋 오늘의 작업 내용

### 1. 주간 정리

- **성과**: STT 모듈 고도화(VAD, 후처리), 번역 모듈(Deep Translator) 구현 완료.
- **상태**: 입력 영상 -> 자막(SRT) -> 번역된 자막(SRT)까지의 파이프라인 구축됨.

### 2. TTS 준비

- 다음 단계인 음성 합성(TTS)을 위해 VALL-E X 리포지토리 클론 및 의존성 설치.
- `requirements.txt` 업데이트.

## 🔧 기술적 진행사항

### Dependencies

- `phonemizer`: 텍스트를 음소(Phoneme)로 변환하는 라이브러리 설치 필수 (VALL-E X 의존성).

| 주간 보고 | 완료 | 완료 | ✅ |
| TTS 환경설정 | 완료 | 완료 | ✅ |

## 🚧 이슈 사항 및 해결 방안

- **Espeak**: `phonemizer` 사용을 위해 시스템에 `espeak-ng` 백엔드 설치 필요. 윈도우 환경 변수 설정 주의.
