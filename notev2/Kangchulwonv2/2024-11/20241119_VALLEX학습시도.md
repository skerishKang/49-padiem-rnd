# AI 기반 다국어 음성 합성 및 실시간 립싱크 더빙 시스템 개발일지

## 📋 오늘의 작업 내용

### 1. 기본 합성 테스트

- 제공된 체크포인트(`checkpoints/`)를 로드하여 텍스트 -> 음성 생성 테스트.
- **결과**: 영어 텍스트 입력 시 자연스러운 영어 음성 생성 확인.

### 2. Zero-shot Cloning 테스트

- 내 목소리 3초 녹음(`prompt.wav`) 후 텍스트 입력.
- **결과**: 내 목소리와 유사한 톤으로 영어를 말하는 음성 생성됨. 신기함.
- **한계**: 한국어 억양이 약간 섞이거나, 발음이 뭉개지는 경우 발생.

## 🔧 기술적 진행사항

from utils.generation import SAMPLE_RATE, generate_audio, preload_models
