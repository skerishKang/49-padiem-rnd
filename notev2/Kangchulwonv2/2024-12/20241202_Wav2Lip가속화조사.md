# AI 기반 다국어 음성 합성 및 실시간 립싱크 더빙 시스템 개발일지

## 📋 오늘의 작업 내용

### 1. 텍스트 정규화 필요성

- STT 결과물이나 번역 결과물에 포함된 숫자, 약어, 기호 등을 TTS가 읽기 좋은 형태로 변환해야 함.
- 예: "12/25" -> "십이월 이십오일" (Korean), "December twenty-fifth" (English).

### 2. 라이브러리 조사

- **Korean**: `kss` (문장 분리), `jamo` (자모 분리). 숫자 읽기는 정규식 기반 커스텀 로직 필요.
- **English**: `num2words` 라이브러리 활용.

## 🔧 기술적 진행사항

### Num2Words

```python
from num2words import num2words
print(num2words(42, lang='en')) # forty-two
```

## 🚧 이슈 사항 및 해결 방안

- **다의어**: "1/2"이 "이분의 일"인지 "일월 이일"인지 문맥 파악 필요. -> 일단 날짜 형식(YYYY-MM-DD) 위주로 처리하고 나머지는 그대로 둠.
