# AI 기반 다국어 음성 합성 및 실시간 립싱크 더빙 시스템 개발일지

## 📋 오늘의 작업 내용

### 1. 문맥 끊김 문제

- 자막을 한 줄씩 번역하면 앞뒤 문맥이 끊겨 오역 발생.
- 예: "It works." -> "그것은 일한다." (문맥상 "작동한다"가 맞음).

### 2. 해결 방안: 슬라이딩 윈도우 (Sliding Window)

- 현재 문장을 번역할 때 앞뒤 문장을 같이 프롬프트로 제공하거나, 덩어리(Chunk)로 묶어서 번역.
- Deep Translator는 문맥 인식이 약하므로, 5~10문장씩 묶어서 `\n`으로 연결해 보내는 방식 채택.

## 🔧 기술적 진행사항

### Chunking Strategy

```python
chunks = ["\n".join(sentences[i:i+5]) for i in range(0, len(sentences), 5)]
# Translate chunks -> Split back to sentences
```
