## 📋 오늘의 작업 내용

### 1. 전처리 파이프라인 구현

- `modules/rvc/preprocess.py` 작성.
- 입력 영상 -> 오디오 추출 -> Demucs(보컬 분리) -> 슬라이싱 -> 정규화 -> 저장.
- 이 과정을 통해 원본 영상만 있으면 자동으로 학습 데이터셋이 생성되도록 자동화.

### 2. 테스트

- 10분짜리 유튜브 인터뷰 영상으로 테스트.
- 약 150개의 깨끗한 음성 세그먼트 생성됨.

def prepare_dataset(video_path, output_dir):
    audio = extract_audio(video_path)
