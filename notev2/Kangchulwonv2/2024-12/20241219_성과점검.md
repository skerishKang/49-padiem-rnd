# AI 기반 다국어 음성 합성 및 실시간 립싱크 더빙 시스템 개발일지

## 📋 오늘의 작업 내용

### 1. 학습 환경 구축

- RVC 학습은 VRAM을 많이 소모하므로, 학습 시에는 다른 프로세스(FastAPI 등)를 종료하는 것이 안전.
- `train_rvc.py` 스크립트 작성.
- 사전 학습된 모델(Pretrained Base Model) 다운로드 (`Hubert`, `f0` 모델 등).

### 2. 의존성 점검

- `fairseq`, `faiss-gpu` 등 필수 라이브러리 설치.
- 윈도우 환경에서 `faiss` 설치 시 이슈(Conda 권장)가 있어 `pip install faiss-gpu`로 해결 시도.

## 🔧 기술적 진행사항

### Requirements

- `pip install fairseq faiss-gpu tensorboard`
