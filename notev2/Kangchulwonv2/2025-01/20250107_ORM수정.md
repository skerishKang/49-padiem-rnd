# AI 기반 다국어 음성 합성 및 실시간 립싱크 더빙 시스템 개발일지

## 📋 오늘의 작업 내용

### 1. 정량 평가 (Speaker Verification)

- `resemblyzer` 라이브러리를 사용하여 원본 화자와 변환된 음성의 임베딩 유사도(Cosine Similarity) 측정.
- **목표**: 0.75 이상.
- **결과**: 평균 0.78 달성. (동일 인물로 인식될 수준).

### 2. 정성 평가 (청취 테스트)

- 팀원 대상 블라인드 테스트.
- "원본과 비슷하다"는 의견 다수.

## 🔧 기술적 진행사항

### Resemblyzer

```python
from resemblyzer import VoiceEncoder, preprocess_wav
encoder = VoiceEncoder()
embed1 = encoder.embed_utterance(preprocess_wav("original.wav"))
embed2 = encoder.embed_utterance(preprocess_wav("converted.wav"))
similarity = np.dot(embed1, embed2)
```
