# AI 기반 다국어 음성 합성 및 실시간 립싱크 더빙 시스템 개발일지

## 📋 오늘의 작업 내용

### 1. 연동 테스트

- 한국어 텍스트 -> 영어 번역 -> VALL-E X (영어 생성) -> RVC (내레이터 음색 입히기).
- **결과**: VALL-E X 단독 사용 시보다 훨씬 더 원본 화자와 비슷한 목소리가 나옴. 특히 억양은 영어인데 목소리는 한국인인 독특한 효과 달성.

### 2. 문제점

- VALL-E X가 가끔 노이즈를 생성하면 RVC가 그 노이즈까지 증폭시킴.
- 중간에 노이즈 필터(Denoiser) 추가 필요성 검토.

## 🔧 기술적 진행사항

### Observation

- Hybrid approach works best for cross-lingual dubbing.

## 🚧 이슈 사항 및 해결 방안
