# AI 기반 다국어 음성 합성 및 실시간 립싱크 더빙 시스템 개발일지

## 📋 오늘의 작업 내용

### 1. A/B 테스트

- **Case A**: VALL-E X (Zero-shot) 단독.
- **Case B**: VALL-E X + RVC.
- **평가**: 팀원 5명 대상 청취 테스트.
- **결과**: 5명 전원 Case B 선호. "목소리가 더 선명하고 원본과 같다"는 평.

### 2. 노이즈 필터 적용

- `noisereduce` 라이브러리 적용.
- TTS 생성 직후 노이즈 제거 수행.
- **결과**: RVC 변환 후 들리던 미세한 잡음(Hissing) 감소.

## 🔧 기술적 진행사항

### Noise Reduction

```python
import noisereduce as nr
reduced_noise = nr.reduce_noise(y=audio, sr=rate)
```
