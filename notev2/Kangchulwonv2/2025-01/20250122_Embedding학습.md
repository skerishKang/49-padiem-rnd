# AI 기반 다국어 음성 합성 및 실시간 립싱크 더빙 시스템 개발일지

## 📋 오늘의 작업 내용

### 1. 배치 처리 (Batch Processing)

- 문장 단위로 TTS/RVC를 호출하는 대신, 여러 문장을 리스트로 모아서 한 번에 처리.
- 모델 로딩 오버헤드를 줄일 수 있음.
- VALL-E X는 배치 추론을 공식 지원하지 않으나, 루프를 돌리더라도 모델을 유지한 채로 처리하면 효율적.

### 2. 구현

- `generate_batch(texts, ...)` 메서드 추가.

## 🔧 기술적 진행사항

### Batch Logic

- 모델 로드 -> 문장 1 처리 -> 문장 2 처리 ... -> 모델 언로드.
