# AI 기반 다국어 음성 합성 및 실시간 립싱크 더빙 시스템 개발일지

## 📋 오늘의 작업 내용

### 1. Wav2Lip 원리 분석

- **목표**: 임의의 오디오에 맞춰 비디오 속 인물의 입모양을 동기화.
- **구조**: Generator(입모양 생성) + Discriminator(품질 판별) + Expert Sync Net(동기화 판별).
- **입력**: 비디오 프레임 + 오디오 세그먼트.

### 2. 라이선스 재확인

- LRS2 데이터셋 라이선스 문제로 상업적 이용 시 주의 필요.
- 현재는 연구 개발 단계이므로 사용 가능하나, 추후 자체 데이터셋 학습 고려.

## 🔧 기술적 진행사항

### Architecture

- SyncNet loss is crucial for accurate lip-sync.
