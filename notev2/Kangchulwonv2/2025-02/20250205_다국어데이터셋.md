# AI 기반 다국어 음성 합성 및 실시간 립싱크 더빙 시스템 개발일지

## 📋 오늘의 작업 내용

### 1. 기본 추론

- `inference.py` 실행.
- 입력: 원본 비디오 + TTS 생성 오디오(영어).
- **결과**: 입모양은 오디오에 맞춰 움직이나, 얼굴 부분이 흐릿하게 나옴 (Blurry Face).

### 2. GAN 모델 테스트

- `wav2lip_gan.pth` 사용.
- **결과**: 입모양 디테일이 조금 더 살아있으나 여전히 해상도 차이로 인한 이질감 존재.

## 🔧 기술적 진행사항

### Command

```bash
python inference.py --checkpoint_path checkpoints/wav2lip_gan.pth --face input.mp4 --audio input.wav
```
