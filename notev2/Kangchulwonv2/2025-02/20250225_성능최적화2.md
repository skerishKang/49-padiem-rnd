# AI 기반 다국어 음성 합성 및 실시간 립싱크 더빙 시스템 개발일지

## 📋 오늘의 작업 내용

### 1. 싱크 보정

- 생성된 오디오 길이와 원본 비디오 길이가 다를 때(번역 후 말이 길어지거나 짧아짐), Wav2Lip은 오디오 길이에 맞춰 비디오를 자르거나 멈춤.
- **해결**: 원본 비디오의 프레임을 오디오 길이에 맞춰 늘리거나 줄이는(Time Stretching) 로직 추가.
- `ffmpeg -itsscale` 옵션 활용.

### 2. 결과 확인

- 입모양과 소리가 정확하게 일치함.

## 🔧 기술적 진행사항

### FFmpeg

```bash
ffmpeg -i input.mp4 -filter:v "setpts=PTS/SPEED" ...
```
