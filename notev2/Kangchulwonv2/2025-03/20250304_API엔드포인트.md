# AI 기반 다국어 음성 합성 및 실시간 립싱크 더빙 시스템 개발일지

## 📋 오늘의 작업 내용

### 1. 파일 업로드 API

- `POST /upload`: 비디오 파일을 받아 `data/inputs`에 저장.
- `UploadFile` 타입 사용. 대용량 파일 처리를 위해 `shutil.copyfileobj` 활용.

### 2. 작업 요청 API

- `POST /process`: 업로드된 파일 ID와 타겟 언어를 받아 작업 시작.
- 현재는 동기(Synchronous) 방식으로 구현 (테스트용). 추후 비동기로 전환 예정.

## 🔧 기술적 진행사항

### Endpoint

```python
@router.post("/upload")
async def upload_file(file: UploadFile):
    # save file
    return {"filename": file.filename}
```
