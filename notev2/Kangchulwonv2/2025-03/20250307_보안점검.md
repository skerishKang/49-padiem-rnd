# AI 기반 다국어 음성 합성 및 실시간 립싱크 더빙 시스템 개발일지

## 📋 오늘의 작업 내용

### 1. 보안 점검

- **AWS WAF**: SQL Injection, XSS 공격 차단 규칙 적용.
- **Security Group**: 최소 권한 원칙 적용 (80, 443 포트만 개방, SSH는 Bastion Host 통해서만 허용).
- **S3 Bucket Policy**: 퍼블릭 액세스 차단 및 CloudFront를 통해서만 접근 허용.

### 2. 주간 성과 정리

- **성과**:
  - 모바일 웹 UI 최적화 완료.
  - AWS 클라우드 인프라 구축 및 오토스케일링 검증 완료.
  - 부하 테스트를 통해 동시 접속 50명 처리 능력 확인.
- **차주 목표**:
  - 공개 베타 런칭 준비 (랜딩 페이지, 이용 약관).
  - 마케팅 활동 시작.

## 🔧 기술적 진행사항

### S3 Bucket Policy (CloudFront OAC)

```json
{
    "Version": "2012-10-17",
    "Statement": {
        "Sid": "AllowCloudFrontServicePrincipal",
        "Effect": "Allow",
        "Principal": {
            "Service": "cloudfront.amazonaws.com"
        },
        "Action": "s3:GetObject",
        "Resource": "arn:aws:s3:::padiem-bucket/*",
        "Condition": {
            "StringEquals": {
                "AWS:SourceArn": "arn:aws:cloudfront::...:distribution/..."
            }
        }
    }
}
```
