# AI 기반 다국어 음성 합성 및 실시간 립싱크 더빙 시스템 개발일지

## 📋 오늘의 작업 내용

### 1. 랜딩 페이지 기획

- **구성**:
  - Hero Section: 서비스 소개 영상 (데모) 및 "무료 체험하기" 버튼.
  - Features: 실시간 립싱크, 다국어 지원, 고품질 음성 합성 등 핵심 기능 소개.
  - Use Cases: 유튜버, 교육 영상, 기업 홍보물 등 활용 사례.
  - Pricing: 요금제 안내 (Beta 기간 무료).

### 2. 개발 환경

- **프레임워크**: Next.js (SEO 최적화 및 빠른 로딩 속도).
- **호스팅**: Vercel (CI/CD 연동 용이).
- **디자인**: Tailwind CSS (빠른 스타일링).

## 🔧 기술적 진행사항
