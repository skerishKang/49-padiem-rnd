# AI 기반 다국어 음성 합성 및 실시간 립싱크 더빙 시스템 개발일지

## 📋 오늘의 작업 내용

### 1. 법적 고지 사항 작성

- **이용 약관**: 서비스 이용 범위, 저작권 귀속(사용자에게 귀속), 금지 행위(딥페이크 악용 등) 명시.
- **개인정보 처리방침**: 수집 항목(이메일, 영상 데이터), 보유 기간(탈퇴 시 즉시 파기), 제3자 제공 동의 등.
- **책임 제한**: AI 생성물의 정확성에 대한 면책 조항 포함.

### 2. 랜딩 페이지 UI 구현

- **반응형 디자인**: 모바일/태블릿/데스크탑 대응.
- **애니메이션**: Framer Motion을 사용하여 스크롤 시 요소가 부드럽게 나타나도록 구현.

## 🔧 기술적 진행사항

### Framer Motion 적용

```javascript
import { motion } from "framer-motion"

<motion.div
  initial={{ opacity: 0, y: 20 }}
  whileInView={{ opacity: 1, y: 0 }}
  transition={{ duration: 0.5 }}
>
  <h2>Feature Title</h2>
</motion.div>
```
