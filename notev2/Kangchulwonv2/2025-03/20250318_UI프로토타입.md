# AI 기반 다국어 음성 합성 및 실시간 립싱크 더빙 시스템 개발일지

## 📋 오늘의 작업 내용

### 1. 기본 컴포넌트 구현

- `st.file_uploader`: 파일 업로드.
- `st.selectbox`: 언어 선택.
- `st.button`: 처리 시작.

### 2. API 연동

- `requests.post`로 백엔드 `/upload` 및 `/process` 호출.
- 응답받은 `job_id`를 세션 상태(`st.session_state`)에 저장.

## 🔧 기술적 진행사항

### Code

```python
uploaded = st.file_uploader("Video")
if uploaded and st.button("Start"):
    # API Call
    st.session_state['job_id'] = response['job_id']
```
