# AI 기반 다국어 음성 합성 및 실시간 립싱크 더빙 시스템 개발일지

## 📋 오늘의 작업 내용

### 1. 요금제 설계 및 UI

- **Free**: 월 5분 무료 (워터마크 포함).
- **Pro**: 월 29,000원 / 60분 (워터마크 제거, 우선 처리).
- **Enterprise**: 별도 문의 (API 제공, 전용 서버).
- **UI**: 비교 테이블(Comparison Table) 형태로 혜택 명시.

### 2. 샌드박스 테스트

- **시나리오**:
  - 카드 결제 성공 -> 크레딧 즉시 충전 확인.
  - 잔액 부족 실패 -> 에러 메시지 출력 확인.
  - 결제 취소(환불) -> 크레딧 회수 확인.
