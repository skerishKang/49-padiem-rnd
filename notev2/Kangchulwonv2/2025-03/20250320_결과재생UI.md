# AI 기반 다국어 음성 합성 및 실시간 립싱크 더빙 시스템 개발일지

## 📋 오늘의 작업 내용

### 1. 결과 재생

- 작업 완료 후 반환된 결과 파일 경로(URL)를 `st.video`에 연결.
- 브라우저에서 바로 재생 가능.

### 2. 다운로드 버튼

- `st.download_button` 구현.
- 바이너리 데이터를 읽어서 다운로드 제공.

## 🔧 기술적 진행사항

### Download

```python
with open(result_path, "rb") as f:
    st.download_button("Download", f, file_name="dubbed.mp4")
```

## 🚧 이슈 사항 및 해결 방안

- **파일 서빙**: 로컬 파일 경로를 직접 브라우저가 읽을 수 없음. -> Streamlit의 `static` 폴더 기능을 쓰거나, FastAPI에서 정적 파일 서빙(`StaticFiles`) 설정 후 URL로 접근해야 함. FastAPI 방식 채택.
