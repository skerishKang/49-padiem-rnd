# AI 기반 다국어 음성 합성 및 실시간 립싱크 더빙 시스템 개발일지

## 📋 오늘의 작업 내용

### 1. 관리자 기능 정의

- **대시보드**: 일별 매출, 신규 가입자, 활성 사용자(DAU/MAU) 그래프.
- **사용자 관리**: 회원 정보 조회, 크레딧 수동 지급/차감, 강제 탈퇴.
- **결제 관리**: 결제 내역 조회, 환불 처리.

### 2. 개발 환경

- **프레임워크**: `Streamlit` (기존 코드 재사용).
- **접근 제어**: `admin` 권한을 가진 계정만 접속 가능하도록 미들웨어 설정.

## 🔧 기술적 진행사항

### 관리자 페이지 권한 체크

```python
if not st.session_state.user.is_admin:
```
