# AI 기반 다국어 음성 합성 및 실시간 립싱크 더빙 시스템 개발일지

## 📋 오늘의 작업 내용

### 1. 작업 목록 UI 개선

- **문제**: 리스트가 길어지면 로딩이 느리고 찾기 어려움.
- **해결**:
  - 페이지네이션(Pagination) 적용 (10개씩 보기).
  - 검색 및 필터(날짜, 상태, 언어) 기능 추가.
  - 썸네일 미리보기 제공.

### 2. 다운로드 만료 처리

- **정책**: 서버 스토리지 비용 절감을 위해 원본/결과물은 7일 후 자동 삭제.
- **UI**: 만료된 파일은 다운로드 버튼 비활성화 및 "만료됨" 표시.

## 🔧 기술적 진행사항

### S3 Lifecycle Rule

- 버킷 설정에서 객체 생성 후 7일 뒤 자동 삭제되도록 규칙 적용.
- DB에는 메타데이터만 남기고 `is_expired` 플래그 업데이트.

## 📊 진행 상황

|------|------|------|------|
| UI 개선 | 완료 | 완료 | ✅ |
| 만료 정책 적용 | 완료 | 완료 | ✅ |

## 🚧 이슈 사항 및 해결 방안

- **복구 요청**: 사용자가 만료된 파일 복구를 요청하는 경우. -> 정책상 불가함을 약관에 명시하고, 삭제 전(1일 전) 알림 발송.
