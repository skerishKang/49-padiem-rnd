# AI 기반 다국어 음성 합성 및 실시간 립싱크 더빙 시스템 개발일지

## 📋 오늘의 작업 내용

### 1. 4월 상세 일정

- **Week 1**: 모의 해킹 및 취약점 조치.
- **Week 2**: 워터마킹 모듈(Invisible Watermark) 개발.
- **Week 3**: B2B API Gateway 구축.
- **Week 4**: HiFi-GAN Vocoder 파인튜닝.

### 2. 모의 해킹 시나리오

- **계정 탈취**: Brute Force, Credential Stuffing.
- **권한 상승**: 일반 유저가 관리자 API 호출 시도.
- **서비스 거부**: 대량의 요청으로 서버 마비 시도 (Rate Limit 동작 확인).

## 🔧 기술적 진행사항

### Rate Limit 테스트

- `ab` (Apache Bench) 도구로 초당 100회 요청 전송 -> 429 Too Many Requests 응답 확인.
