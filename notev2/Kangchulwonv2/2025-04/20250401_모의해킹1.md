# AI 기반 다국어 음성 합성 및 실시간 립싱크 더빙 시스템 개발일지

## 📋 오늘의 작업 내용

### 1. 취약점 점검 (OWASP Top 10)

- **Injection**: SQL Injection 시도 -> ORM 사용으로 방어됨 확인.
- **Broken Authentication**: 무작위 대입 공격(Brute Force) -> 5회 실패 시 계정 잠금 기능 정상 작동.
- **Sensitive Data Exposure**: HTTPS 미적용 구간 없음 확인.

### 2. 발견된 취약점

- **XSS (Cross Site Scripting)**: 게시판 댓글 입력란에 스크립트 태그(`<script>`) 실행 가능성 발견.
- **조치**: 입력값 살균(Sanitization) 라이브러리 `bleach` 적용.

## 🔧 기술적 진행사항

### XSS 방어 코드

```python
import bleach

def clean_input(text):
    return bleach.clean(text, tags=[], attributes={}, strip=True)
```
