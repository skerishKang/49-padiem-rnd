# AI 기반 다국어 음성 합성 및 실시간 립싱크 더빙 시스템 개발일지

## 📋 오늘의 작업 내용

### 1. 필요성

- STT 및 번역 결과가 100% 완벽할 수 없음.
- 사용자가 직접 수정할 수 있는 인터페이스(Human-in-the-loop)가 필수적.
- 수정된 자막을 바탕으로 TTS를 다시 생성해야 함.

### 2. UI 기획

- **좌측**: 원본 영상 플레이어.
- **우측**: 자막 편집 리스트 (Timecode, Original Text, Translated Text).
- **기능**: 텍스트 수정 후 'Regenerate TTS' 버튼 클릭 시 해당 구간만 다시 생성.

## 🔧 기술적 진행사항

### Workflow

- Edit Text -> Update DB -> Call Partial TTS API -> Merge Audio -> Update Video.
