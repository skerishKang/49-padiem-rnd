# AI 기반 다국어 음성 합성 및 실시간 립싱크 더빙 시스템 개발일지

## 📋 오늘의 작업 내용

### 1. Streamlit Data Editor

- `st.data_editor`를 사용하여 판다스 DataFrame(자막 데이터)을 엑셀처럼 편집 가능하게 구현.
- 컬럼: `Start`, `End`, `Source`, `Target`.

### 2. 상태 연동

- 편집 후 'Save' 버튼을 누르면 변경된 데이터가 백엔드로 전송됨.

## 🔧 기술적 진행사항

### UI Code

```python
edited_df = st.data_editor(df, num_rows="dynamic")
if st.button("Save Changes"):
    update_subtitles(edited_df)
```
