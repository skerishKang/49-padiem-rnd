# AI 기반 다국어 음성 합성 및 실시간 립싱크 더빙 시스템 개발일지

## 📋 오늘의 작업 내용

### 1. 워터마킹 기술 조사

- **Visible Watermark**: 로고나 텍스트를 영상 위에 덧씌움. (무료 플랜용)
- **Invisible Watermark**: 주파수 영역(DCT/DFT)에 정보를 숨김. 육안으로는 보이지 않으나 전용 디코더로 식별 가능. (유료 플랜용, 딥페이크 악용 추적)

### 2. 라이브러리 선정

- **영상**: `ffmpeg` (Visible), `blind-watermark` (Invisible).
- **오디오**: `audiowmark` (오디오 주파수 대역 확산 방식).

## 🔧 기술적 진행사항

### Invisible Watermark (Python)

```python
from blind_watermark import WaterMark

bwm = WaterMark(password='padiem_secret')
bwm.read_img('frame.png')
bwm.read_wm('user_id_1234')
bwm.embed('output_frame.png')
```
