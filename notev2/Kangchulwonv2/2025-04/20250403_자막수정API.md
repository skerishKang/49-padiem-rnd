# AI 기반 다국어 음성 합성 및 실시간 립싱크 더빙 시스템 개발일지

## 📋 오늘의 작업 내용

### 1. 수정 API

- `POST /jobs/{job_id}/subtitles`: 수정된 자막 리스트를 받아 저장.
- 변경된 세그먼트만 식별하여 `needs_regeneration` 플래그 설정.

### 2. 부분 재생성 (Partial Regeneration)

- 전체 파이프라인을 다시 돌리는 대신, 변경된 문장만 TTS+RVC 수행.
- 기존 오디오 트랙의 해당 구간을 묵음 처리하고 새 오디오 삽입(Overlay).

## 🔧 기술적 진행사항

### Logic

```python
original_audio = AudioSegment.from_file(...)
new_segment = generate_tts(new_text)
final_audio = original_audio.overlay(new_segment, position=start_ms)
```
