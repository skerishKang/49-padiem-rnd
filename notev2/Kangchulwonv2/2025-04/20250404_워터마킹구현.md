# AI 기반 다국어 음성 합성 및 실시간 립싱크 더빙 시스템 개발일지

## 📋 오늘의 작업 내용

### 1. 워터마킹 모듈 구현

- **Visible**: `ffmpeg` 필터 체인에 `overlay` 필터 추가.
- **Invisible**: 영상 처리 파이프라인 마지막 단계(Wav2Lip 직후)에 삽입 로직 추가. 사용자 ID를 바이너리로 변환하여 삽입.

### 2. 주간 성과 정리

- **성과**:
  - 모의 해킹을 통해 XSS, IDOR 등 주요 취약점 조치 완료.
  - 워터마킹 기술 도입으로 AI 윤리 및 보안성 강화.
- **차주 목표**:
  - B2B API Gateway 구축.
  - API 문서화 (Swagger/Redoc).

## 🔧 기술적 진행사항

### FFmpeg Overlay

```bash
ffmpeg -i input.mp4 -i logo.png -filter_complex "overlay=W-w-10:H-h-10" output.mp4
```
