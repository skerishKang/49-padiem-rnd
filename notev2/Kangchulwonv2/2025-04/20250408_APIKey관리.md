# AI 기반 다국어 음성 합성 및 실시간 립싱크 더빙 시스템 개발일지

## 📋 오늘의 작업 내용

### 1. API Key 발급 로직

- **생성**: `uuid4` 기반의 고유 키 생성.
- **저장**: DB에 해시(Hash)하여 저장 (보안).
- **Kong 연동**: 키 생성 시 Kong Admin API를 호출하여 Consumer 및 Key 등록.

### 2. Kong 플러그인 설정

- **Key Auth**: 헤더에 `apikey`가 포함되어야 요청 허용.
- **Rate Limiting**: 초당 5회, 분당 100회로 제한.

## 🔧 기술적 진행사항

### Kong Admin API 호출 (Python)

```python
def register_consumer(username, apikey):
    # Consumer 생성
    requests.post(f"{KONG_ADMIN_URL}/consumers", json={'username': username})
    # Key 등록
    requests.post(f"{KONG_ADMIN_URL}/consumers/{username}/key-auth", json={'key': apikey})
```
