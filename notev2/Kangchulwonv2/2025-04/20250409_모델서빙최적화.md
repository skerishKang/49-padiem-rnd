## 📋 오늘의 작업 내용

### 1. TorchScript 검토

- PyTorch 모델을 직렬화하고 최적화하여 C++ 런타임 등에서 실행 가능하게 함.
- VALL-E X와 RVC 모델 변환 시도.
- **결과**: 일부 커스텀 연산(Custom Ops) 지원 문제로 변환 실패. 난이도 높음.

### 2. 대안: ONNX Runtime

- ONNX 변환이 더 범용적.
- 하지만 현재 단계에서 모델 변환까지 하기엔 리스크가 큼.
- **결정**: 일단 Python 런타임 최적화(JIT Compile 등)에 집중하고 모델 변환은 보류.
