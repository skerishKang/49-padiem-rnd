# AI 기반 다국어 음성 합성 및 실시간 립싱크 더빙 시스템 개발일지

## 📋 오늘의 작업 내용

### 1. 음질 개선 필요성

- 현재 VALL-E X의 기본 Vocoder는 약간의 기계음(Metallic Sound)과 잡음(Hissing)이 섞여 있음.
- 방송급 품질을 위해 HiFi-GAN Vocoder를 자체 데이터로 파인튜닝 필요.

### 2. 데이터셋 구성

- **소스**: VCTK (영어), AI-Hub (한국어), JVS (일본어), AISHELL (중국어).
- **선별**: SNR 40dB 이상의 고품질 오디오만 필터링.
- **전처리**: 24kHz, 16-bit PCM 변환, Mel-spectrogram 생성.

## 🔧 기술적 진행사항

### Mel-spectrogram 변환

```python
mel = librosa.feature.melspectrogram(y=wav, sr=24000, n_fft=1024, hop_length=256, n_mels=80)
log_mel = np.log10(mel + 1e-5)
```

## 🚧 이슈 사항 및 해결 방안

- **데이터 양**: Vocoder 학습에는 수백 시간의 데이터가 필요하지 않음. 화자 다양성보다는 음질 자체가 중요. -> 각 언어별 상위 10명 화자 데이터만 사용 (총 20시간).
