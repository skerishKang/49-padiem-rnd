# AI 기반 다국어 음성 합성 및 실시간 립싱크 더빙 시스템 개발일지

## 📋 오늘의 작업 내용

### 1. Sentry 도입

- 실시간 에러 추적을 위해 Sentry(Self-hosted or Cloud) 연동.
- FastAPI 미들웨어 및 Celery 시그널에 Sentry SDK 적용.
- **효과**: 에러 발생 시 즉시 알림 및 문맥 정보(입력값, 환경변수 등) 확인 가능.

### 2. 테스트

- 고의로 예외 발생시켜 Sentry 대시보드에 잡히는지 확인.

## 🔧 기술적 진행사항

### Integration

```python
import sentry_sdk
sentry_sdk.init(dsn="...", traces_sample_rate=1.0)
```
