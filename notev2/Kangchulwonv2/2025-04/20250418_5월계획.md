## 🔧 기술적 진행사항

### DB 스키마 설계 (Team)

```python
class Team(Base):
    __tablename__ = "teams"
    id = Column(Integer, primary_key=True)
    name = Column(String)
    owner_id = Column(Integer, ForeignKey("users.id"))
    members = relationship("User", secondary="team_members")
```
