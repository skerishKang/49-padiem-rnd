## 📋 오늘의 작업 내용

### 1. 팀 대시보드 UI (Streamlit)

- **구성**:
  - **멤버 목록**: 이름, 이메일, 역할(Role), 가입일.
  - **사용량 통계**: 팀 전체의 API 호출 횟수 및 스토리지 사용량 시각화.
  - **설정**: 팀 이름 변경, 멤버 추방, 권한 수정.

### 2. RBAC (Role-Based Access Control) 적용

- **Middleware**: API 요청 시 사용자의 역할을 확인하여 접근 제어.
- **테스트**: Viewer 권한으로 프로젝트 삭제 시도 -> `403 Forbidden` 확인.

## 🔧 기술적 진행사항

### RBAC 데코레이터

```python
def require_role(role: str):
    def decorator(func):
        @wraps(func)
        def wrapper(*args, **kwargs):
            if current_user.role != role:
                raise HTTPException(status_code=403)
            return func(*args, **kwargs)
        return wrapper
    return decorator
```
