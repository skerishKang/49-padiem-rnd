# AI 기반 다국어 음성 합성 및 실시간 립싱크 더빙 시스템 개발일지

## 📋 오늘의 작업 내용

### 1. Transfer Acceleration 활성화

- **개념**: 사용자가 가장 가까운 Edge Location으로 업로드하면, AWS 내부 고속 네트워크를 통해 S3로 전송됨.
- **설정**: S3 버킷 속성에서 활성화. 비용 발생함.

### 2. 업로드 테스트

- **환경**: 1GB 영상 파일 업로드.
- **결과**:
  - 일반 업로드: 4분 20초.
  - 가속 업로드: 2분 10초 (약 50% 단축).

## 🔧 기술적 진행사항

### Boto3 설정

```python
s3_client = boto3.client('s3', region_name='ap-northeast-2', config=Config(s3={'use_accelerate_endpoint': True}))
```
