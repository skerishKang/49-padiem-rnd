# AI 기반 다국어 음성 합성 및 실시간 립싱크 더빙 시스템 개발일지

## 📋 오늘의 작업 내용

### 1. 마케팅 자동화 필요성

- 회원 가입 후 이탈하는 사용자를 잡기 위한 리텐션 마케팅 필요.
- 웰컴 이메일, 사용 팁, 구독 유도 등 자동화된 캠페인 실행.

### 2. 도구 비교

- **HubSpot**: 기능 강력, CRM 연동 우수, 비쌈.
- **Mailchimp**: 이메일 중심, UI 직관적, 중저가.
- **AWS SES + Custom**: 저렴하지만 직접 개발 필요.
- **결정**: 초기에는 **AWS SES**를 활용하여 직접 구축하고, 추후 HubSpot 도입 고려. (비용 절감 및 데이터 내재화)
