# AI 기반 다국어 음성 합성 및 실시간 립싱크 더빙 시스템 개발일지

## 📋 오늘의 작업 내용

### 1. 6월 목표 설정

- **주제**: 모바일 앱(Android/iOS) 프로토타입 개발.
- **이유**: 크리에이터들의 모바일 접근성 요구 증가. 웹앱(PWA)으로는 성능 한계 존재.
- **기술 스택**: Flutter (Cross-platform).

### 2. 상세 일정

- **Week 1**: Flutter 프로젝트 세팅 및 UI 설계.
- **Week 2**: API 연동 및 로그인/회원가입 구현.
- **Week 3**: 영상 업로드 및 더빙 요청 기능.
- **Week 4**: 결과물 재생 및 다운로드, 내부 테스트.

## 🔧 기술적 진행사항

### Flutter 아키텍처 (GetX)

- 상태 관리 및 라우팅을 위해 GetX 패턴 채택.
- **구조**: Model - View - Controller.
