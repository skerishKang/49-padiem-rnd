# AI 기반 다국어 음성 합성 및 실시간 립싱크 더빙 시스템 개발일지

## 📋 오늘의 작업 내용

### 1. UI 구조

- **Splash Screen**: 로고 애니메이션.
- **Login**: 이메일/비밀번호 입력, 소셜 로그인 버튼.
- **Home**: 내 프로젝트 목록 (Card View), 새 프로젝트 생성 버튼(+).
- **Detail**: 영상 재생, 언어 선택, 더빙 시작 버튼.

### 2. 레이아웃 구현

- `Scaffold`, `AppBar`, `BottomNavigationBar` 등 기본 위젯 활용.
- `GetX`를 사용하여 페이지 간 이동 구현.

## 🔧 기술적 진행사항

### 라우팅 설정

```dart
GetMaterialApp(
  initialRoute: '/',
  getPages: [
    GetPage(name: '/', page: () => Home()),
    GetPage(name: '/login', page: () => Login()),
  ],
)
```
