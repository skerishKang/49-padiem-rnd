# AI 기반 다국어 음성 합성 및 실시간 립싱크 더빙 시스템 개발일지

## 📋 오늘의 작업 내용

### 1. 통합 파이프라인 아키텍처 설계

- **목표**: 개별적으로 동작하던 모듈(STT, TTS, RVC, Wav2Lip)을 하나의 웹 인터페이스에서 순차적으로 실행.
- **구조**:
  - **Frontend**: Streamlit을 활용한 직관적인 대시보드.
  - **Backend**: FastAPI를 이용한 RESTful API 서버.
  - **Orchestrator**: 작업 순서 관리 및 상태 모니터링 (`job_manager.py`).

### 2. 기술 스택 확정

- **Web Framework**: Streamlit (빠른 프로토타이핑 및 데이터 시각화 용이).
- **API Server**: FastAPI (비동기 처리 및 Swagger 문서 자동화).
- **Inter-process Communication**: HTTP Requests.

## 🔧 기술적 진행사항

### API 설계

- `/api/audio/extract`: 오디오 추출.
- `/api/stt/transcribe`: 음성 인식 (Whisper).
- `/api/tts/generate`: 음성 합성 (VALL-E X).
- `/api/rvc/convert`: 목소리 변환.
- `/api/lipsync/run`: 립싱크 생성.
