# AI 기반 다국어 음성 합성 및 실시간 립싱크 더빙 시스템 개발일지

## 📋 오늘의 작업 내용

### 1. 번역 모듈

- `backend/routers/text.py`: `deep-translator` 패키지 활용.
- **기능**: STT 결과 텍스트를 타겟 언어(영어, 일본어, 중국어 등)로 번역.
- **UI**: 원문과 번역문을 나란히 보여주고 수정 가능하도록 `st.data_editor` 활용.

### 2. TTS 모듈 (VALL-E X)

- `backend/routers/tts.py`: VALL-E X 모델을 이용한 다국어 음성 합성.
- **특징**: 3초 분량의 레퍼런스 오디오만 있으면 목소리 복제 가능 (Zero-Shot).
- **통합**: STT 단계에서 추출된 오디오를 레퍼런스로 자동 지정.

## 🔧 기술적 진행사항

### VALL-E X

- Cross-lingual TTS 기능을 통해 한국어 화자의 목소리로 영어/일본어 발화 생성.
