# AI 기반 다국어 음성 합성 및 실시간 립싱크 더빙 시스템 개발일지

## 📋 오늘의 작업 내용

### 1. End-to-End 테스트

- **시나리오**: 1분 분량의 한국어 뉴스 영상 업로드 -> 영어로 더빙 -> RVC 적용 -> 립싱크 -> 결과 확인.
- **결과**: 전체 프로세스 소요 시간 약 3분. 결과물 품질 양호.

### 2. 버그 수정

- **경로 문제**: 윈도우 환경에서 파일 경로 역슬래시(`\`) 처리 문제 수정 (`pathlib` 활용).
- **세션 초기화**: Streamlit 새로고침 시 작업 상태 날아가는 문제. -> `st.session_state` 및 로컬 파일(`session.json`)에 상태 저장 로직 추가.

## 🔧 기술적 진행사항

### Pathlib

```python
from pathlib import Path
input_path = Path("data") / "inputs" / "video.mp4"
```
