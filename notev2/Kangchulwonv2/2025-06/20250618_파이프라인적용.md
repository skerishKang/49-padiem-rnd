## 📋 오늘의 작업 내용

### 1. 예외 처리 강화

- `backend/job_manager.py`: 작업 중 에러 발생 시 `traceback`을 로그에 남기고 작업 상태를 `Failed`로 변경.
- **재시도 로직**: 일시적인 오류(네트워크 등) 발생 시 3회 자동 재시도.

### 2. 리소스 정리

- 작업 완료 후 임시 파일(`tmp/`) 자동 삭제 로직 추가.
- 서버 디스크 공간 확보를 위해 주기적인 클린업 스케줄러 검토.

## 🔧 기술적 진행사항

### Try-Except Block

```python
try:
    process_pipeline(job_id)
except Exception as e:
    logger.error(f"Job {job_id} failed: {e}")
    job.status = "Failed"
    job.error_message = str(e)
finally:
    cleanup_temp_files(job_id)
```
