# AI 기반 다국어 음성 합성 및 실시간 립싱크 더빙 시스템 개발일지

## 📋 오늘의 작업 내용

### 1. 설문 결과 분석

- **만족도**: 5점 만점에 평균 4.2점.
- **장점**: "신기하다", "웹에서 바로 되니 편하다".
- **단점**: "속도가 느리다", "입모양이 가끔 어색하다".

### 2. 개선 방향 도출

- **속도**: GPU 업그레이드 또는 모델 경량화 필요.
- **품질**: Wav2Lip의 해상도(96x96) 한계. -> Super Resolution(GFPGAN) 적용 필수.

## 🔧 기술적 진행사항

### GFPGAN

- Wav2Lip 후처리 단계에 GFPGAN을 적용하여 얼굴 해상도를 높이는 옵션 기본 활성화 결정.

## 🚧 이슈 사항 및 해결 방안

- **처리 시간 증가**: GFPGAN 적용 시 처리 시간 2배 증가. -> 사용자에게 "고화질(느림)" vs "일반(빠름)" 선택권 부여.
