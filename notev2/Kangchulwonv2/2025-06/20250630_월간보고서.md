# AI 기반 다국어 음성 합성 및 실시간 립싱크 더빙 시스템 개발일지

## 📋 오늘의 작업 내용

### 1. 6월 월간 보고서

- **주요 내용**: 모바일 앱 런칭, 초고속 음성 복제 기술 확보, 베타 테스트 결과.
- **결론**: 기술적 완성도는 높으나, 번역 품질과 사용자 편의성(편집 기능) 보완이 필요함.

### 2. 7월 계획 수립

- **Week 1**: 사용자 사전(Glossary) 기획 및 DB 설계.
- **Week 2**: 사용자 사전 기능 구현 (프론트/백엔드).
- **Week 3**: 구간 편집(Segment Editing) 기능 개발.
- **Week 4**: Llama 3 기반 번역 엔진 도입.

## 🔧 기술적 진행사항

### 7월 아키텍처 변경 예고

- 번역 모듈을 기존 DeepL API 호출 방식에서 로컬 LLM Serving(vLLM) 방식으로 변경 예정.
