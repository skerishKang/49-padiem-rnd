## 📋 오늘의 작업 내용

### 1. GFPGAN 도입 배경

- 6월 베타 테스트 결과, Wav2Lip의 출력 해상도(96x96)가 낮아 전체 영상 품질을 저하시킨다는 지적.
- 얼굴 복원(Face Restoration) 모델인 GFPGAN을 후처리 단계에 추가하여 선명도 개선 결정.

### 2. 모듈 연동

- `backend/routers/lipsync.py`: Wav2Lip 수행 후 GFPGAN 추론 실행 옵션 추가.
- **방식**: Wav2Lip 결과 프레임 -> 얼굴 감지 -> GFPGAN 복원 -> 원본 프레임에 합성.

## 🔧 기술적 진행사항

### GFPGAN Inference

```python
restorer = GFPGANer(model_path='experiments/pretrained_models/GFPGANv1.4.pth', upscale=1)
_, restored_faces, _ = restorer.enhance(img, has_aligned=False, only_center_face=False, paste_back=True)
```
