# AI 기반 다국어 음성 합성 및 실시간 립싱크 더빙 시스템 개발일지

## 📋 오늘의 작업 내용

### 1. 기능 정의

- 사용자가 특정 단어(고유명사, 전문용어)의 번역을 미리 지정할 수 있는 기능.
- 프로젝트별 또는 사용자별 전역 설정 가능.

### 2. DB 스키마 설계

- **테이블**: `glossaries`, `glossary_terms`.
- **Glossary**: 사용자 ID, 이름, 소스 언어, 타겟 언어.
- **Term**: 용어집 ID, 원문(Source Term), 번역문(Target Term).

## 🔧 기술적 진행사항

### SQLAlchemy 모델

```python
class Glossary(Base):
    __tablename__ = "glossaries"
    id = Column(Integer, primary_key=True)
    user_id = Column(Integer, ForeignKey("users.id"))
    name = Column(String)
    source_lang = Column(String)
    target_lang = Column(String)
    terms = relationship("GlossaryTerm", back_populates="glossary")

class GlossaryTerm(Base):
    __tablename__ = "glossary_terms"
    id = Column(Integer, primary_key=True)
    glossary_id = Column(Integer, ForeignKey("glossaries.id"))
    source_term = Column(String)
    target_term = Column(String)
```
