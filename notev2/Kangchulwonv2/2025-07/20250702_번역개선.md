# AI 기반 다국어 음성 합성 및 실시간 립싱크 더빙 시스템 개발일지

## 📋 오늘의 작업 내용

### 1. 번역 API 교체

- 기존 `deep-translator` (Google Translate 웹 크롤링 방식)의 불안정성 및 품질 이슈.
- **대안**: Google Cloud Translation API (v2) 정식 연동 검토.
- **테스트**: 주요 문장 100개 번역 비교. 정식 API가 문맥 파악 및 고유명사 처리에서 우수함.

### 2. UI 개선

- 번역 결과 수정 기능 강화.
- **기능**: 사용자가 번역된 텍스트를 직접 수정하면, 수정된 텍스트로 TTS를 생성하도록 로직 변경.

## 🔧 기술적 진행사항

### Google Cloud Translation

```python
from google.cloud import translate_v2 as translate
translate_client = translate.Client()
result = translate_client.translate(text, target_language=target)
```
