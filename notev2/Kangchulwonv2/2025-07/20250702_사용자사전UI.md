## 📋 오늘의 작업 내용

### 1. 관리 UI (Streamlit)

- **목록**: 내 용어집 리스트 표시.
- **상세**: 용어 추가/수정/삭제 테이블 에디터.
- **업로드**: CSV 파일 드래그 앤 드롭으로 일괄 등록.

### 2. CSV 파싱 로직

- `pandas`를 사용하여 CSV 파일 읽기.
- 유효성 검사 (필수 컬럼 존재 여부, 중복 체크).

## 🔧 기술적 진행사항

### Streamlit Data Editor

```python
edited_df = st.data_editor(df, num_rows="dynamic")
if st.button("저장"):
    save_terms(glossary_id, edited_df)
```
