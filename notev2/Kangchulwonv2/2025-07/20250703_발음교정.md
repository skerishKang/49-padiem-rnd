# AI 기반 다국어 음성 합성 및 실시간 립싱크 더빙 시스템 개발일지

## 📋 오늘의 작업 내용

### 1. 발음 문제 분석

- 한국어 화자의 목소리로 영어 합성 시, "R/L" 발음이나 억양이 한국어처럼 들리는 현상 (Korean Accent).
- **원인**: VALL-E X의 Cross-lingual 기능이 원본 화자의 억양 정보를 강하게 유지하기 때문.

### 2. 해결 방안 모색

- **Prompt Engineering**: 레퍼런스 오디오를 원어민의 것으로 교체하면 발음은 좋아지지만 목소리가 바뀜.
- **절충안**: 같은 성별의 원어민 레퍼런스 오디오를 사용하여 TTS 생성 후, RVC로 원래 화자의 목소리를 입히는 방식 채택.

## 🔧 기술적 진행사항

### Hybrid Pipeline

1. **TTS**: 원어민 레퍼런스 -> VALL-E X -> 원어민 발음의 오디오 생성.
2. **RVC**: 생성된 오디오(Source) + 원본 화자 목소리(Target) -> 원본 화자 목소리로 변환된 원어민 발음 오디오.
