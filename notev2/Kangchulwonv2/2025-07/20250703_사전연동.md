# AI 기반 다국어 음성 합성 및 실시간 립싱크 더빙 시스템 개발일지

## 📋 오늘의 작업 내용

### 1. DeepL Glossary 생성

- 사용자가 등록한 용어집을 DeepL API를 통해 서버에 업로드하고 `glossary_id`를 발급받음.
- **제약**: DeepL은 지원하는 언어 쌍이 제한적임 (예: 한국어-영어 가능, 한국어-중국어 불가).

### 2. 번역 요청 시 적용

- 번역 API 호출 시 `glossary_id` 파라미터 포함.
- **Fallback**: DeepL이 지원하지 않는 언어 쌍의 경우, 번역 전/후처리를 통해 단순 치환(Replace) 방식으로 구현.

## 🔧 기술적 진행사항

### DeepL API 호출

```python
glossary = translator.create_glossary(
    name,
    source_lang="KO",
    target_lang="EN",
    entries={"PADIEM": "PADIEM", "강철원": "Cheolwon Kang"}
)
result = translator.translate_text("강철원은 PADIEM의 개발자입니다.", glossary=glossary)
```
