### 2. 구간 편집 기획

- **UI**: Waveform(파형) 시각화 및 드래그 앤 드롭으로 구간 선택.
- **백엔드**: `ffmpeg`를 사용하여 해당 구간만 잘라내어(Trim) 처리 후 원본 영상에 다시 합치기(Merge).

## 🔧 기술적 진행사항

### FFmpeg Trim & Merge

```bash
# 1. 구간 자르기
ffmpeg -i input.mp4 -ss 00:00:10 -t 00:00:05 -c copy segment.mp4

# 2. 더빙 후 합치기 (Concat Demuxer)
ffmpeg -f concat -i list.txt -c copy output.mp4
```
