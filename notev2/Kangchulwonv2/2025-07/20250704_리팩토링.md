# AI 기반 다국어 음성 합성 및 실시간 립싱크 더빙 시스템 개발일지

## 📋 오늘의 작업 내용

### 1. 코드 구조 개선

- `backend/routers/` 내의 비즈니스 로직을 `backend/services/`로 분리.
- **목적**: 라우터는 요청/응답 처리만 담당하고, 실제 처리는 서비스 계층에서 수행하여 재사용성 증대.

### 2. 설정 파일 통합

- 하드코딩된 경로 및 설정값들을 `config.yaml`로 통합 관리.
- 환경 변수(`.env`)를 통해 API 키 관리 보안 강화.

## 🔧 기술적 진행사항

### Service Layer Pattern

```python
# routers/tts.py
@router.post("/generate")
async def generate_tts(request: TTSRequest):
    return await tts_service.generate(request)
```
