# AI 기반 다국어 음성 합성 및 실시간 립싱크 더빙 시스템 개발일지

## 📋 오늘의 작업 내용

### 1. 배경 소음 문제

- 야외 촬영 영상 등에서 배경 소음이 섞여 있으면 STT 인식률 저하 및 음성 복제 품질 저하 발생.
- **해결**: Meta의 `Demucs` 모델을 사용하여 목소리(Vocals)와 배경음(Drums, Bass, Other) 분리.

### 2. 모듈 연동

- `backend/routers/audio.py`: 업로드 직후 Demucs 실행 옵션 추가.
- **프로세스**: 원본 오디오 -> Demucs -> Vocals Only -> STT/TTS 레퍼런스로 사용.

## 🔧 기술적 진행사항

### Demucs

```python
subprocess.run(["demucs", "-n", "htdemucs", input_audio])
```

- 분리된 `vocals.wav`를 이후 파이프라인의 입력으로 사용.
