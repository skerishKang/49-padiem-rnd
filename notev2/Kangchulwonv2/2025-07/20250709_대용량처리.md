# AI 기반 다국어 음성 합성 및 실시간 립싱크 더빙 시스템 개발일지

## 📋 오늘의 작업 내용

### 1. 대용량 영상 테스트

- **대상**: 10분 길이의 다큐멘터리 영상 (FHD).
- **문제**: Wav2Lip 처리 중 OOM(Out Of Memory) 발생.
- **원인**: 전체 프레임을 한 번에 메모리에 로드하려고 시도함.

### 2. 청크(Chunk) 처리 구현

- 영상을 1분 단위로 잘라서(Segmenting) 순차적으로 처리하고, 마지막에 다시 합치는(Concatenating) 방식 도입.
- **장점**: 메모리 사용량을 일정하게 유지 가능.
- **단점**: 전체 처리 시간은 약간 증가하지만 안정성 확보.

## 🔧 기술적 진행사항

### Video Segmentation

```python
# ffmpeg segment muxer
subprocess.run(["ffmpeg", "-i", input, "-f", "segment", "-segment_time", "60", "out%03d.mp4"])
```
