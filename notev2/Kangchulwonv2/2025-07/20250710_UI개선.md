# AI 기반 다국어 음성 합성 및 실시간 립싱크 더빙 시스템 개발일지

## 📋 오늘의 작업 내용

### 1. 드래그 앤 드롭 업로드

- Streamlit `file_uploader` 위젯 활용.
- 여러 파일을 동시에 업로드할 수 있도록 `accept_multiple_files=True` 설정.

### 2. 작업 목록 관리

- 업로드된 파일들을 리스트 형태로 보여주고, 각 파일별로 상태(대기, 진행중, 완료) 표시.
- **일괄 처리**: 여러 파일을 선택하고 '일괄 더빙 시작' 버튼을 누르면 순차적으로 작업 큐에 등록.

## 🔧 기술적 진행사항

### Batch Processing UI

```python
uploaded_files = st.file_uploader("Choose videos", accept_multiple_files=True)
if st.button("Process All"):
    for file in uploaded_files:
        job_manager.add_job(file)
```
