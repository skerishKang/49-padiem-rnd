# AI 기반 다국어 음성 합성 및 실시간 립싱크 더빙 시스템 개발일지

## 📋 오늘의 작업 내용

### 1. 프롬프트 설계

- **역할 부여**: "당신은 전문 영상 번역가입니다."
- **제약 조건**: "구어체로 자연스럽게 번역하세요.", "존댓말을 사용하세요.", "사용자 사전을 참고하세요."
- **Few-shot**: 예시 3개를 제공하여 스타일 유도.

### 2. 품질 비교 (vs DeepL)

- **문맥 파악**: Llama 3가 앞뒤 문맥을 더 잘 이해하고 자연스러운 한국어 구사.
- **고유명사**: 사용자 사전을 프롬프트에 포함시키면 Llama 3가 훨씬 더 유연하게 적용함.
- **속도**: DeepL API가 더 빠름. (Llama 3는 토큰 생성 시간 소요).

## 🔧 기술적 진행사항

### Prompt Template

```python
prompt = f"""
You are a professional video translator. Translate the following text from {source_lang} to {target_lang}.
Use the provided glossary: {glossary_dict}
Style: Colloquial, Polite.

Text: {input_text}
Translation:
"""
```

## 🚧 이슈 사항 및 해결 방안

- **환각(Hallucination)**: 가끔 엉뚱한 말을 덧붙임. -> `temperature`를 0.1로 낮추고 `stop_token` 설정.
