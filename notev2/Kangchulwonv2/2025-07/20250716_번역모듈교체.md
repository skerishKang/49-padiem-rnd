# AI 기반 다국어 음성 합성 및 실시간 립싱크 더빙 시스템 개발일지

## 📋 오늘의 작업 내용

### 1. 모듈 교체

- `TextProcessor` 클래스 내의 `translate` 메서드를 수정.
- 기존 `deepl_client` 호출 부분을 `openai_client` (vLLM 호환) 호출로 변경.
- **설정**: `config.yaml`에서 번역 엔진 선택 가능하도록 (`deepl` vs `llm`).

### 2. 비동기 처리

- LLM 응답 속도를 고려하여 `asyncio`를 활용한 비동기 호출 적용.
- 여러 문장을 동시에 번역 요청 (Batch Processing)하여 처리량 증대.

## 🔧 기술적 진행사항

### OpenAI Client (vLLM)

```python
client = AsyncOpenAI(api_key="EMPTY", base_url="http://localhost:8000/v1")
response = await client.chat.completions.create(
    model="beomi/Llama-3-Open-Ko-8B",
    messages=[{"role": "user", "content": prompt}],
    temperature=0.1
)
```
