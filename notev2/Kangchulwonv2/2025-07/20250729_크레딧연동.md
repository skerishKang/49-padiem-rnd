# AI 기반 다국어 음성 합성 및 실시간 립싱크 더빙 시스템 개발일지

## 📋 오늘의 작업 내용

### 1. 차감 로직

- 더빙 작업 요청 시 예상 소요 시간(분) 계산.
- 사용자 잔여 크레딧 확인.
- **부족 시**: 작업 거부 및 충전 페이지 안내.
- **충분 시**: 크레딧 차감(Hold) 후 작업 시작. 실패 시 환불(Rollback).

### 2. 알림 시스템

- 크레딧 잔액이 10% 미만일 때 경고 알림.
- 결제 실패 시 "결제 정보 업데이트 요망" 이메일 발송.

## 🔧 기술적 진행사항

### Transaction

```python
try:
    user.credits -= cost
    db.commit()
    start_dubbing_job()
except Exception:
    db.rollback()
    raise HTTPException(status_code=400, detail="Insufficient credits")
```
