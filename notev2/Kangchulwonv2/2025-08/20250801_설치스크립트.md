# AI 기반 다국어 음성 합성 및 실시간 립싱크 더빙 시스템 개발일지

## 📋 오늘의 작업 내용

### 1. 배포 편의성 개선

- 복잡한 Python 환경 설정 및 의존성 설치 과정을 자동화.
- **Windows**: `setup.bat` 및 `run.bat` 스크립트 작성.
- **Linux**: `setup.sh` 및 `run.sh` 스크립트 작성.

### 2. 가상환경 자동 구성

- 스크립트 실행 시 `venv` 생성 -> `pip install -r requirements.txt` -> `ffmpeg` 경로 확인 -> 실행.
- 비개발자도 원클릭으로 실행 가능하도록 구성.

## 🔧 기술적 진행사항

### Batch Script

```batch
@echo off
if not exist ".venv" (
    python -m venv .venv
)
call .venv\Scripts\activate
pip install -r backend/requirements.txt
python -m uvicorn backend.main:app --reload
```
