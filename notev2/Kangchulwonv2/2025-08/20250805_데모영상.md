# AI 기반 다국어 음성 합성 및 실시간 립싱크 더빙 시스템 개발일지

## 📋 오늘의 작업 내용

### 1. 시연 녹화

- OBS Studio를 사용하여 전체 파이프라인 실행 과정 녹화.
- **구성**: 인트로 -> 파일 업로드 -> 옵션 설정 -> 처리 과정(로그) -> 결과 재생 -> 아웃트로.
- **내레이션**: 개발자가 직접 기능 설명 더빙 (본인 목소리 -> RVC로 아나운서 톤 변환하여 사용).

### 2. 편집

- 불필요한 대기 시간(Loading) 배속 처리.
- 주요 기능에 자막 및 하이라이트 효과 추가.

## 🔧 기술적 진행사항

### Dogfooding

- 데모 영상의 내레이션 더빙을 **자체 개발한 시스템**을 사용하여 제작. 품질 검증과 홍보 효과 동시 달성.

## 📊 진행 상황

| 항목 | 계획 | 실제 | 상태 |
|------|------|------|------|
| 화면 녹화 | 완료 | 완료 | ✅ |
| 편집 및 완성 | 완료 | 완료 | ✅ |

## 🚧 이슈 사항 및 해결 방안

- **용량**: 완성된 영상 용량이 큼. -> HandBrake로 압축하여 웹 게시 용이하게 변환.
