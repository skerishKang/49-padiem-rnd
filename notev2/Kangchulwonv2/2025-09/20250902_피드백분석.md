# AI 기반 다국어 음성 합성 및 실시간 립싱크 더빙 시스템 개발일지

## 📋 오늘의 작업 내용

### 1. 피드백 분류

- **채널**: 채널톡, 이메일, 앱스토어 리뷰, 디스코드.
- **유형**:
  - **버그**: 15% (로그인 풀림, 업로드 실패 등).
  - **기능 요청**: 50% (자막 스타일 변경, 배경음 조절, 팀 기능).
  - **기타**: 35% (가격 문의, 칭찬).

### 2. 우선순위 선정

- **P0 (즉시)**: 로그인 세션 만료 문제 (사용자 불편 큼).
- **P1 (이번 주)**: 배경음 볼륨 조절 슬라이더 추가.
- **P2 (다음 달)**: 팀 협업 기능.

## 🔧 기술적 진행사항

### JWT Refresh Token

- 로그인 풀림 문제는 Access Token 만료 시 Refresh Token으로 갱신하는 로직이 프론트엔드에서 가끔 실패하는 버그. -> `Dio` Interceptor 수정하여 갱신 로직 견고화.
