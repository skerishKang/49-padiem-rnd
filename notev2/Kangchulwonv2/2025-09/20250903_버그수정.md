# AI 기반 다국어 음성 합성 및 실시간 립싱크 더빙 시스템 개발일지

## 📋 오늘의 작업 내용

### 1. 로그인 세션 수정

- Refresh Token 만료 시(보통 2주) 로그아웃 처리하고 로그인 페이지로 리다이렉트.
- Access Token 만료 시(1시간) 백그라운드에서 조용히 갱신(Silent Refresh).

### 2. 배경음 조절 UI

- 더빙 옵션 페이지에 '배경음 볼륨' 슬라이더(0% ~ 100%) 추가.
- 기본값은 80%.

## 🔧 기술적 진행사항

### Audio Mixing

- `pydub` 또는 `ffmpeg` 사용 시 볼륨 파라미터 적용.

```python
bgm = bgm - 10 # -10dB
final = voice.overlay(bgm)
```
