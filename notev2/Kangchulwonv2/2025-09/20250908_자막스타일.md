# AI 기반 다국어 음성 합성 및 실시간 립싱크 더빙 시스템 개발일지

## 📋 오늘의 작업 내용

### 1. 기능 정의

- **폰트**: 기본 폰트 외에 나눔고딕, 배달의민족 등 인기 무료 폰트 제공.
- **스타일**: 글자 색상, 외곽선(Stroke), 배경색(Box), 위치(상/중/하) 조절.
- **미리보기**: 설정 변경 시 실시간으로 미리보기 제공.

### 2. 구현 (FFmpeg)

- `drawtext` 필터를 사용하여 자막 스타일 적용.
- `.srt` 파일을 `.ass` (Advanced Substation Alpha) 형식으로 변환하여 스타일 입히는 방식도 고려했으나, `drawtext`가 더 간단하여 채택.

## 🔧 기술적 진행사항

### FFmpeg Drawtext

```bash
ffmpeg -i input.mp4 -vf "drawtext=fontfile=NanumGothic.ttf:text='안녕하세요':fontcolor=white:fontsize=24:x=(w-text_w)/2:y=h-50" output.mp4
```
