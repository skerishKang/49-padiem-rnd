# AI 기반 다국어 음성 합성 및 실시간 립싱크 더빙 시스템 개발일지

## 📋 오늘의 작업 내용

### 1. i18n 적용

- **라이브러리**: `react-i18next` (웹), `flutter_localizations` (앱).
- **언어**: 한국어(ko), 영어(en), 일본어(ja).
- **리소스**: JSON 파일로 언어별 문자열 관리.

### 2. 언어 감지

- 브라우저/기기 설정 언어를 자동 감지하여 초기 언어 설정.
- 헤더에 언어 변경 드롭다운 메뉴 제공.

## 🔧 기술적 진행사항
