# AI 기반 다국어 음성 합성 및 실시간 립싱크 더빙 시스템 개발일지

## 📋 오늘의 작업 내용

### 1. API 캐싱

- **대상**: 자주 변하지 않는 데이터 (공지사항, 요금제 정보, 사용자 프로필).
- **도구**: Redis + FastAPI Cache.
- **정책**: TTL 1시간 설정. 데이터 변경 시 캐시 무효화(Invalidate).

### 2. 정적 자원 최적화

- **이미지**: WebP 포맷으로 변환하여 용량 50% 절감.
- **JS/CSS**: Minify & Gzip 압축 적용.
- **CDN**: CloudFront 캐시 정책 최적화 (Cache-Control 헤더 설정).

## 🔧 기술적 진행사항

### FastAPI Cache

```python
@app.get("/notices")
@cache(expire=3600)
async def get_notices():
    return await db.fetch_all(...)
```
