# AI 기반 다국어 음성 합성 및 실시간 립싱크 더빙 시스템 개발일지

## 📋 오늘의 작업 내용

### 1. 기능 정의

- **팀 생성**: 사용자가 팀을 만들고 소유자(Owner)가 됨.
- **멤버 초대**: 이메일로 초대 링크 발송.
- **권한 관리**: Owner(모든 권한), Editor(편집 가능), Viewer(보기만 가능).
- **공유 워크스페이스**: 팀원끼리 영상 및 자막 공유.

### 2. DB 설계

- **Teams**: 팀 ID, 이름, 소유자 ID.
- **TeamMembers**: 팀 ID, 사용자 ID, 역할(Role).

## 🔧 기술적 진행사항

### RBAC (Role-Based Access Control)

- API 호출 시 사용자의 팀 내 역할을 확인하여 권한 제어.

```python
def check_permission(user, team, required_role):
    member = get_member(user, team)
    if member.role < required_role:
        raise Forbidden()
```
