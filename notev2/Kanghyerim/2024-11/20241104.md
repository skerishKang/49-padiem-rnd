- **अ젠다**: 사용자 매뉴얼·IRIS 입력 가이드 개편, 교육 콘텐츠 제작, 버전 관리·배포

## 1. 연구 배경·목적 (책임 관점)

- IRIS 리허설 결과를 반영해 **사용자 매뉴얼·입력 가이드**를 최신 상태로 개편하고, 교육 영상·자료를 정리하여 현장 연구자들이 스스로 IRIS 제출을 처리할 수 있게 하는 것이 목표.
- 접근성(WCAG 2.1)과 보안 규정(산업부 가이드)을 동시에 만족하는 문서 구조를 확립해야 함.

---

## 2. 오늘 주요 진행 내용

### 2.1 매뉴얼 구조 리팩터링
- 기존 장별 구성에서, 실사용 흐름에 맞춘 4단계 구조로 재편:
  1) 준비
  2) 입력
  3) 검증
  4) 제출·사후 관리
- 각 단계마다 체크포인트와 "자주 묻는 질문" 블록을 추가해 교육·실무에서 재사용 가능하도록 설계.

### 2.2 IRIS 입력 가이드 업데이트
- IRIS 포털 UI 스크린샷 36장을 최신 빌드 기준으로 교체.
- 필드별 입력 규칙(길이, 형식, 한/영 병기, 금지 문자 등)을 표 형태로 정리.
- Markdown→HTML 변환 시 발생하는 서식 깨짐 이슈를 교정.

### 2.3 교육 콘텐츠 제작
- 15분 분량 화면 녹화 기반 교육 영상 2편 제작.
  - IRIS 입력 전체 플로우 데모.
  - 자동 업로드 스크립트 사용 예시.
- 한글 자막과 다국어 자막 스크립트를 함께 작성해, 이후 번역·배포를 고려.

### 2.4 버전 관리·배포
- SharePoint에 `iris_user_guide_v1.2.pdf` 업로드, 변경 이력(Changelog) 작성.
- 배포 대상 27명에게 Teams 공지 및 링크 일괄 발송.
