## 2. 오늘 주요 진행 내용

### 2.1 12월 실행 계획 세분화
- 업로드 백오프 개선, 교육 ETL, Safari 13 대응, 중국 CDN 로그 고도화 등 과제를 2~3주 스프린트 단위로 쪼개어 정리.
- 각 작업에 담당자·예상 일정·리스크를 매핑.

### 2.2 온보딩 체크리스트 정리
- 외부/신규 인력용 온보딩 체크리스트를 12개 항목(계정·권한·보안 교육·IRIS 프로세스·콘솔 사용법 등)으로 구성.
- SharePoint Forms와 연동해 각 항목에 대해 완료 체크·서명 흐름을 만들도록 설계.

### 2.3 교육 데이터 ETL 설계
- LMS API → Data Lake → KPI 대시보드로 이어지는 데이터 파이프라인 구상.
- Airflow DAG `training_attendance_sync` 초안을 작성해, 교육 참석/미참석 데이터를 자동 반영할 수 있도록 준비.

### 2.4 커뮤니케이션
- 실행 계획과 온보딩 체크리스트 초안을 Slack PM 채널에 공유하고, 피드백 4건을 반영 예정으로 기록.
