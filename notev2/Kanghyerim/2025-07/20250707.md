# 연구노트 (책임자 관점)

- **날짜**: 2025-07-07
- **작성자**: 강혜림
- **역할**: 책임연구원
- **진행 단계**: 1단계 분기 기초 연구·계획 수립
- **아젠다**: Sprint W2 중간 점검, 교육 보고 경량화, Dark Mode 피드백 반영

---

## 1. 연구 배경·목적 (책임 관점)

- 7월 Sprint W2 중간 시점에서 번다운·실험 상태를 점검하고, 교육 보고 파일 용량을 줄여 실사용성을 높이는 것이 목적.
- Dark Mode 도입 후 들어온 피드백을 반영해 문서·UI 품질을 보완하고자 함.

---

## 2. 오늘 주요 진행 내용

### 2.1 Sprint W2 중간 점검
- 번다운 기준 약 **30SP** 소화.
- Dark Mode 관련 실험 티켓 1건 진행 중.

### 2.2 교육 보고 경량화
- 교육 보고 파일 용량을 18MB → 9MB로 축소.
- 이미지 압축·폰트 조정으로 가독성은 유지하면서 파일 크기만 절반 수준으로 감소.

### 2.3 Dark Mode 피드백 반영
- 사내 사용자 피드백을 바탕으로 색상·폰트·간격 일부 조정 계획 수립.
- 캡션·레이블이 모바일에서 줄바꿈되는 이슈는 7/8 CSS 레벨에서 수정하기로 합의.

### 2.4 커뮤니케이션
- Sprint 중간 점검·보고 경량화·Dark Mode 피드백 내용 공유.

---

## 3. 산출물·지표

- Sprint 번다운 30SP.
- 경량화된 교육 보고 파일(9MB).
- Dark Mode 개선 항목 리스트.

---

## 4. 책임자 관점 의사결정·리스크

- **결정**
  - 교육 보고는 항상 10MB 이하 용량을 목표로 유지.

- **리스크**
  - 일부 구형 브라우저에서 폰트·캡션 표시 이슈 남아 있어, 7/8 이후 추가 CSS 보완이 필요.
