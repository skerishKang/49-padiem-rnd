## 2. 검증 방법론 (Methodology)

### 2-1. 정량 지표 측정 기준
| 지표명 | 측정 라이브러리 | 수식/방법 |
| :--- | :--- | :--- |
| **WER** (Word Error Rate) | `jiwer` | $WER = \frac{S + D + I}{N}$ (S:교체, D:삭제, I:삽입, N:전체단어) |
| **BLEU** | `sacrebleu` | $n$-gram 정밀도의 기하평균 기반 번역 유사도 측정 |
| **PSNR** | `torchmetrics` | $10 \cdot \log_{10}(\frac{MAX^2}{MSE})$ (영상의 신호 대 잡음비) |
| **FID** | `clean-fid` | Inception Network 특징 공간에서의 Fréchet 거리 측정 |

### 2-2. 테스트 데이터셋 (Gold Standard)
- **한국어 (KO)**: KSS Dataset 중 발화 명확성이 검증된 100개 샘플
- **영어 (EN)**: LJSpeech Dataset 중 무작위 추출된 100개 샘플
